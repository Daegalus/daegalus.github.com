So I spent a good portion of my night trying to get Windows 8 to work on the Macbook Air. In the end I finally succeeded and it works amazingly well.

So I will give quick and dirty instructions, they won't be too detailed, but hopefully those that are doing this are knowledgable enough to figure this out part way. You will need 2 USBs for this process, and access to your OSX partition, along with patience.

  1. I recommend premtively downloading the Orca MSI editor and also making a Bootcamp Driver USB for all the drivers (the driver USB is required.) You will also need an archiver. Put all of these on 1 USB , just make sure you put the non Bootcamp stuff on there AFTER you make it a Bootcamp Driver USB from OSX.

  2. In OSX, make a Bootcamp Driver USB and set it aside.

  3. From another Windows install on another computer, or another partition, make a Bootable USB Installer for Windows 8. You can use the Windows 7 DVD/USB Download Tool or any other means.

  4. Run the Windows Installer from USB like you would do for Windows 7. The install process is almost 100% identical.

  5. Once it reboots, make sure that after you select Windows in the Mac bootloader, not to touch your keyboard. I have had it lockup a few times and had to do fresh installs. Just let it do its thing until you hit the License agreement.

  6. Now you will notice that the Screen is cut in half, and there are 2 copies of hte top half of your screen. This sadly is due to a buggy Windows 8 Intel HD 3000 driver. You will have to click randomly on the bottom of the screen till it switches to the next screen. This is pretty annoying. I tried tabbing through this, but I have no clue if there is a "Checkbox" or not down there. I just finally got it to move on.

  7. This problem will persist through a while, so you will have to use your knowledge of tabbing and windows shortcuts to move around, and a bit of mental image of how things can/are laid out.

  8. Once you get it all setup and boot in, you should be able to hit the Explorer shortcut or the Desktop tile.

  9. Once there plugin the USB or however you are storing the Orca editor, and install it onto the system.

  10. Once done, plug in the USB with Bootcamp Drivers.

  11. Open Orca, and browse into the "F:\WindowsSupport\Drivers\Apple" (Driver letter doesn't matter, I am just picking something). Select the BootCamp or Bootcamp64.msi and open it.

  12. In the Tables column, scrolldown until you see "LaunchConditions" and select it.

  13. Look for a row that says "VersionNT=601" right-click it and drop row. If you have problems with it not seeing Mac partitions or not figuring out the system model properly, delete the last line with "SYSTEMMODELNAME" in the line.

  14. Save it.

  15. Now go to the folder, and run the MSI file directly. Wait till it does all its installs and so on.

  16. Now go back up one folder to WindowsSupport\Drivers and Go into every folder, and install all the drivers manually. If you know exactly what drivers you need, instll only those, otherwise just go through all of them, and ignore any "Errors" about missing hardware, or other info. I know for Macbook Airs, you can skip the IDT Sigmatel driver. Also for the time being, skip the Intel945, Intel965 install and, IntelHDGraphicsSetup. Others should install fine. If they say Unsupported OS, defer them till later.

  17. Once that is all done, take your favorite Archiver (WinRar, 7zip, whatever), and extract the IntelHDGraphicsSetup exe. It is a self extracting archive, so it just has the installer inside.

  18. Once extracted go to the "Graphics" folder, and open the file "2430-64p.ini" and scroll down until you see [IntelGfx.NTamd64.6.2] (yours might be a bit different if you are doing this on 32-bit). Under it there is a comment about "no install on Win8". Directly below that you should see the one for 6.0 (win7/vista) copy all the lines below it, and paste them under the 6.2 category. Save it.

  19. Go up a folder, and run the Setup.exe. The this should being installing the Driver. If it asks you about downgrading the version, hit yes. When the popup saying it is unsigned, and so on pops up continue. Your display should fix up almost right away.

  20. From here, do the same process (roughly) to get the other Intel packages that you think you might need to install.

During this process, reboot at every step that you are asked to restart on. Also, you will be doing this with only half your screen. So you will have to do window resizing and moving, launching things from memory, or just guessing positions of things to get things working.

This should hopefully help some of you figure it out. This process can be used for any mac, and your mileage will vary. Some systems might not have to mess around with the screen stuff because it will just work (Nvidia/ATI video cards, I cant say for sure though). You will still have to do the rest for proper drivers.
